# RFC Index: Feature Completion Program

This directory tracks RFCs for the MicroClaw feature-completion program.

## Scope

- Auth and authorization model uplift
- Hooks platform MVP
- Session forking data/API model
- Metrics and tracing naming standard

## RFCs

- `0001-authn-authz-model.md`
- `0002-hooks-event-model.md`
- `0003-session-fork-model.md`
- `0004-metrics-naming.md`

## Status Lifecycle

- `Draft`: open for comments
- `Accepted`: implementation approved
- `In Progress`: at least one implementation PR merged
- `Implemented`: all planned PRs merged and validated
- `Superseded`: replaced by newer RFC

## Review Checklist

- Problem statement is concrete
- Backward compatibility is defined
- Migration and rollback are explicit
- Test strategy and lint/CI gates are explicit
- Security impact is reviewed
